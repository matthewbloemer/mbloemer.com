AC to DC Variable Power Supply

Project Goals

This project's goal is to design an AC to DC variable power supply. The design parameters required the power supply to be externally adjustable, with the operating voltage range to be from 5 V to 10 V. The maximum output current for this power supply is 500 mA. There is short-circuit protection built in, so no components would get damaged if the output terminals are shorted. Since this is a school project for my electronics class, I am not allowed to use a monolithic IC voltage regulator, such as an LT317, to regulate the output voltage.
